GVI: Greenness Visibility Index R package

Sebastian Brinkmann, S.M. Labib
July 2021
Dataset DOI

The GVI R package helps researchers compute the Greenness Visibility Index (GVI) presented by Labib, Huck and Lindley (2021). The GVI is calculated using a Digital Surface Model (DSM), Digital Terrain Model (DTM) and Greenness Raster. GVI is written in C++ to provide fast and light weighted functionality.

Go on GitHub for more information.
